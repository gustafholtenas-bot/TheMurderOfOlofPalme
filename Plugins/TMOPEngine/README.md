# TMOP Engine v0.0.1

Included:

- Runtime plugin
- FTMOPTime
- UTMOPClockSubsystem
- Default real-time loop from 23:13:00 to 23:23:00
- Blueprint events for second changes and loop restarts

## Install

Copy the `Plugins` folder into the root of the Unreal project.

Final path:

`TheMurderOfOlofPalme/Plugins/TMOPEngine/TMOPEngine.uplugin`

Then regenerate Visual Studio project files and build `Development Editor | Win64`.
